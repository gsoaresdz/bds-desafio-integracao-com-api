# GitHub API

## Sobre o projeto
Pequeno app que disponibiliza um formulário para buscar os dados de um usuário do Github, utilizando a API pública do Github.

## Tecnologias utilizadas

### Front-end
- HTML / CSS / TypeScript
- ReactJS

### Frontend web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone 

# instalar dependências
yarn install

# executar o projeto
yarn start
```
